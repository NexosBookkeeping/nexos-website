## Publish with Cloudflare Pages
1. Create a GitHub repository named `nexos-website`.
2. Upload all files from this folder, not the ZIP.
3. Cloudflare > Workers & Pages > Create application > Pages > Connect to Git.
4. Select the GitHub repository.
5. Framework preset: None.
6. Build command: leave blank.
7. Output directory: leave blank or `/`.
8. Deploy.
9. Add custom domains:
   - nexosbookkeeping.com
   - www.nexosbookkeeping.com

## Notes
The contact form currently uses `mailto:info@nexosbookkeeping.com`.
After creating Microsoft 365 email, this address will work.
For a true form without opening the visitor's email app, connect Formspree, Basin, or a Cloudflare Worker later.
